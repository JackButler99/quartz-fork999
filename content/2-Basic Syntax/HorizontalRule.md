# Horizontal Rule
Untuk membuat garis horizontal, dapat digunakan tiga buah atau lebih asterisk ( *** ), dashes ( --- ), atau underscores ( ___ ). 

### Syntax Markdown 
```

***
---
___

```

Output dari syntax di atas adalah sebagai berikut:

***
---
___
