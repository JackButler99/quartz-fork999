# Line Break
Line break dapat dibuat dengan menambahkan dua atau lebih spasi di akhir kalimat.

### Markdown Syntax

```Ini merupakan contoh  ``` 
```penggunaan Line Break  ```

```<p> Ini merupakan contoh <br /> penggunaan Line Break  </p>```
